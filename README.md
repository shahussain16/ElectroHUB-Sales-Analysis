# ElectroHUB-Sales-Analysis
Power BI dashboard analyzing ElectroHUB's sales data from 2020 to 2024.


# Files
- `ElectroHUB_Sales_Analysis.pbix`: Full Power BI report with visuals, DAX calculations, and interactive filters.
- `ElectroHUB_Sales_Template.pbit`: Template file (no data) for reuse with similar datasets.
